# 🧛 Vampiro

## Vampire Survivors-Style Roguelite

*Vampiro* is a fast-paced, action-packed roguelite inspired by *Vampire Survivors*, where you fight to survive endless waves of enemies with automatic attacks and strategic upgrades.

---

## 🎮 Gameplay Overview

* **Auto-attacks**: The player automatically attacks with powerful weapons and abilities
* **Endless Survival**: Face an ever-increasing number of enemies
* **Level-Up System**: Upgrade your character with different abilities
* **Multiple Weapons**: Use attacks like Ice Spear, Tornado, and more
* **Scaling Difficulty**: Enemies increase over time to challenge your survival

---

## 🕹️ Controls

| Action | Key                |
| ------ | ------------------ |
| Move   | `W`, `A`, `S`, `D` |
| Attack | Automatic          |

> Attacks are **automatic** — focus on movement and positioning.

---

## ▶️ How to Play

1. Download the latest `.zip` from the **Releases** section
2. Extract the zip file
3. Run the executable for your platform
4. Survive as long as possible and experiment with different builds

---

## 📦 Installation

* **Windows**: Extract and run `.exe`
* **Linux**: Run `.x86_64` with included `.pck`
* **macOS** (if provided): Open `.app`

> No installation required.

## 🚀 Features

* Auto-attack combat system
* Multiple weapons (Ice Spear, Tornado, etc.)
